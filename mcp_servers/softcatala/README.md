Aquest servidor proporciona dos serveis:

 - Accés al diccionari de sinònims de SoftCatala
 - Accés al corrector de Softcatalà

El primer funciona amb la instal·lació bàsica, pero per usar el segon, cal descarregar-se LanguageTool, seguint els seguents passos

 1. Descarregueu el fitxer `https://internal1.languagetool.org/snapshots/LanguageTool-20260312-snapshot.zip` dins del directori "softcatala" (on és aquest fitxer)
 
 2. Descomprimiu el fitxer zip:
    `unzip LanguageTool-20260312-snapshot.zip`

 3. S'extreurà una carpeta `LanguageTool-6.8-SNAPSHOT`.  Canvieu-ne el nom a `LanguageTool-6.8`

 4. Ja podeu esborrar el fitxer `LanguageTool-20260312-snapshot.zip`, i engegar el servei.
